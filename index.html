The Philosophy

Emacs + Unix + Git + Screen/Tmux, filtered through 60 years of knowledge

The Interface Gets Out of Your Way

Natural language is the interface. No memorizing commands. No looking up flags. Just say what you want.

Terminal as Single Source of Truth

Not the file system API. Not a database. The terminal. What you see in the UI is exactly what you'd see if you typed the command.

Sandboxed Shell Customization

Never touches your system settings. Agents create aliases and functions that only exist inside Shebang!

Reactive, Not Project-Rooted

cd somewhere? File browser updates. Switch branches? Context shifts. The environment follows you.

Knowledge Becomes Infrastructure

Type safety is automatic. Tests are suggested. Security is enforced. Performance is default. The AI has read the books.

Agentic by Design

Not an IDE with AI bolted on. Built from the ground up for AI agents to operate alongside humans.

Claude Code as First-Class Citizen

Integrated at the deepest level. Commands route to Claude Code. The terminal IS the native environment.

Built With Itself

Shebang! is built using Shebang! If the tool can't build itself, it's not good enough. Ultimate dogfooding.

"The interface gets out of your way. The work remains."
